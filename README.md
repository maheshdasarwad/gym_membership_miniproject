# Gym Membership Miniproject

A comprehensive gym membership management system designed to streamline member registration, billing, and facility access.

## Features

- **Member Management**: Register and manage gym members with personal information
- **Membership Plans**: Multiple subscription tiers with flexible durations
- **Billing System**: Automated billing and payment tracking
- **Class Scheduling**: Schedule fitness classes and manage enrollments
- **Attendance Tracking**: Track member attendance and gym usage
- **Reports**: Generate membership and financial reports

## Getting Started

### Prerequisites

- Python 3.8 or higher
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/maheshdasarwad/gym_membership_miniproject.git
cd gym_membership_miniproject
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```
